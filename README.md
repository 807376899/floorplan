# floorplan

静态 HTML/SVG 实验室分布图生成器。

## 当前模型

数据拆成两张表：

- `房间表`：教学楼、楼层、走廊段ID、沿段位置、段侧、门牌、长宽、学院等。
- `楼层表`：教学楼、楼层、走廊段ID、起点X、起点Y、终点X、终点Y、走廊宽等。

楼层不再局限于 L 型或 T 型。每层可以用任意数量、任意方向的走廊线段拼成折线、十字、回字、H 型、放射型等布局；房间只需挂到相应走廊段上。

## 页面能力

- 分别上传房间表和楼层表。
- 分别下载可直接上传的模板。
- 在线编辑房间表和楼层表。
- 按楼栋和楼层查看，支持楼层缩略图。
- 画布默认适配显示，也可切换原始大小。

## 运行

```powershell
python -m http.server 5173
```

访问 `http://localhost:5173`。
