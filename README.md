# floorplan

实验室分布图生成器：一个静态 HTML/SVG 小工具，用来把学校实验室表格数据转换成 JSON，并生成模拟的 2D 教室分布图。

## 迭代 1：测试数据字段

测试数据在 `sample-labs.csv`，表头使用中文字段。页面也兼容对应的英文字段名。核心字段如下：

- `教学楼`
- `楼层`
- `实验室名称`
- `南北侧`：填 `南` 或 `北`
- `前门牌`
- `后门牌`：单门房间可与前门牌相同
- `东到西门牌分布`：支持 `东到西递增` 或 `东到西递减`
- `长`、`宽`：按比例绘制房间
- `所属学院`：用于自动着色

## 迭代 2：CSV/Excel 到 JSON

页面支持上传 `.csv`，也支持在联网时通过 SheetJS CDN 解析 `.xlsx` / `.xls` 的第一个工作表。导入后会标准化字段并在右侧显示 JSON 预览。

## 迭代 3：JSON 到 HTML/SVG 分布图

生成规则：

- 北侧房间绘制在走廊上方，南侧房间绘制在走廊下方。
- SVG 左侧标为西，右侧标为东。
- 如果 `east_to_west_order` 是 `东到西递增`，图中从西到东按门牌号递减排列。
- 如果 `east_to_west_order` 是 `东到西递减`，图中从西到东按门牌号递增排列。
- 不同学院自动分配不同颜色，并生成图例。

## 运行

在本目录启动任意静态服务器，然后打开 `index.html`。例如：

```powershell
python -m http.server 5173
```

然后访问 `http://localhost:5173`。
