# floorplan

静态 HTML/SVG 实验室分布图生成器。

## 当前模型

数据拆成两张表：

- `房间表`：教学楼、楼层、走廊段ID、沿段位置、段侧、门牌、长宽、学院等。`段侧` 使用 `北/南/东/西`。
- `楼层表`：教学楼、楼层、走廊段ID、起点X、起点Y、终点X、终点Y、走廊宽等。

楼层不再局限于 L 型或 T 型。每层可以用任意数量的水平或垂直走廊线段拼成折线、十字、回字、H 型等布局；房间只需挂到相应走廊段上。图纸固定为上北下南。

`沿段位置` 从房间最左侧或最上侧开始计算。例如东西向走廊 `main` 上，房间沿段位置为 `0` 时，房间最左侧与走廊最左侧对齐。

## 页面能力

- 分别上传房间表和楼层表。
- 分别下载可直接上传的模板。
- 在线编辑房间表和楼层表。
- 按楼栋和楼层查看，支持楼层缩略图。
- 画布默认适配显示，也可切换原始大小。

## 运行

```powershell
python -m http.server 5173
```

访问 `http://localhost:5173`。
